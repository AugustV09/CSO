3-a) Los tres componentes clave de GNU/Linux son el Kernel (nucleo), el Shell (interprete de comandos) y el FileSystem.

b) La estructura básica de un sistema operativo consiste en un Núcleo o kernel el cual se encarga de administrar la CPU, la memoria y la E/S, es decir, de hacer que el HW y SW trabajen juntos; el Shell es la interfaz a través de la cual el usuario se comunica con el kernel; el FileSystem consiste en el tipo de organización que el SO tiene de los archivos en cada dispositivo de almacenamiento.
